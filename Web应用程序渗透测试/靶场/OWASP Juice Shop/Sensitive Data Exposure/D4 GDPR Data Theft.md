- 描述：在不使用注入的情况下窃取他人的个人数据
- 数据导出功能可获得用户名、邮件地址、订单与商品评价
    ```POST /rest/user/data-export HTTP/1.1```

- 查看示例解决方案
    查看订单详细信息时
    ```GET /rest/track-order/130f-41aeeca9528d9966 HTTP/1.1```
    返回 JSON 中的邮件地址的元音字符被混淆
    ```"email":"b*nd*r@j**c*-sh.*p"```
    - 或在历史订单记录中也可看到
    - 注册可导致相同混淆的邮件地址 
    此处为 ```bondor@jooco-sh.op```
    进入订单历史功能即可查看相同混淆邮件用户的订单
    导出数据通过挑战
