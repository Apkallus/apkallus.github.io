- 描述：揭发 http://127.0.0.1:3000/#/contact 一个藏匿在商店里，却臭名昭著的角色。(提及角色的确切名字)
- 提示：这个挑战的有效难度很大程度上取决于你选择什么工具来应对它。
- 查看示例解决方案：
    - 在“关于我们”页面中可发现 ```5.png``` 是唯一使用 ```.png```格式的文件，下载此文件，进行调查
    - 示例在线图像检查工具无法探测到内容
    - 使用客户端检测工具 OpenStego
    - 安装 Java 环境后使用此工具提取图像
    - 使用在线图片搜索工具确定此图像为 《瑞克和莫蒂》 中的角色```Pickle Rick```
